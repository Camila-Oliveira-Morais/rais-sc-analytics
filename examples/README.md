# Exemplos Sintéticos

Esta pasta contém exemplos pequenos e sintéticos apenas para ilustrar:

- formato das saídas;
- nomes de colunas;
- tipo de conteúdo esperado;
- leitura rápida por pessoas que ainda não possuem os microdados reais.

## Importante

Os arquivos desta pasta:

- não representam resultados oficiais da RAIS;
- não devem ser usados para análise substantiva;
- existem apenas para documentação e onboarding.

## Arquivos disponíveis

- `sample_remuneracao_media_ano_industrial_sc_cbo_ocupacao_2024.xlsx`
- `sample_validacao.xlsx`
- `sample_remuneracao_media_ano_industrial_sc_cbo_ocupacao_2024.csv`
- `sample_validacao.csv`

## Qual formato usar

Prefira abrir os arquivos `.xlsx`.

Motivo:

- o Excel preserva a formatação numérica;
- reduz erro de interpretação de separador decimal;
- evita problemas de importação por configuração regional.

Os arquivos `.csv` foram mantidos apenas como referência simples e intercâmbio técnico.
